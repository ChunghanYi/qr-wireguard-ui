## backend C++ agent for wireguard-ui
```
This web-agentd(C++ version) is used to communicate with wireguard-ui.
```

## How to build for NanoPi(arm64)
```
<Ubuntu 22.04 LTS>
$ ./build_arm64.sh
$ cd build
$ ls -l web-agentd
```
